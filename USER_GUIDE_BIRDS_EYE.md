## Troubleshooting

**Q: I don't see the button**
- A: Make sure you've tapped a satellite first (green box should appear)
- A: Scroll to top of bottom sheet - button is at the very top

**Q: Button doesn't work**
- A: Make sure satellite is fully selected (bottom sheet fully expanded)
- A: Try tapping button again

**Q: Animation is laggy**
- A: Close and reopen the view
- A: This is normal on some devices during first load
